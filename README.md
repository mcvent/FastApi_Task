# FastAPI Blog API

Блог-платформа с JWT авторизацией, постами, категориями, локациями и комментариями. Полностью контейнеризированное приложение на FastAPI с SQLite базой данных.

## Требования и установка

### Системные требования
- Docker (версия 20.10 или выше)
- Docker Compose (версия 2.0 или выше)
- Git (для клонирования репозитория)

### Запуск

```bash
# Клонировать репозиторий
git clone <your-repo-url>
cd FastApi_Task5

# Создать файл с переменными окружения
cp .env.example .env

# Отредактировать .env (сменить SECRET_KEY)
# Отредактировать .env (сменить путь к DB, если необходимо)

# Запустить проект
docker-compose up -d --build

# Проверить работу
curl http://127.0.0.1:8000/docs
